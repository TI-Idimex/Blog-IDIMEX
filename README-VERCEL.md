# BLOG IDIMEX - Publicacao Vercel

Esta pasta contem a versao estatica do BLOG IDIMEX pronta para publicar no Vercel.

## Configuracao recomendada no Vercel

- Framework Preset: Other
- Build Command: deixar vazio
- Output Directory: deixar vazio ou `.`
- Install Command: deixar vazio

## Como publicar

1. Crie um repositorio no GitHub, GitLab ou Bitbucket.
2. Envie todo o conteudo desta pasta para o repositorio.
3. No Vercel, clique em Add New Project.
4. Importe o repositorio.
5. Confirme as configuracoes acima e publique.

## Depois da publicacao

No painel do Vercel, adicione o dominio desejado em Project Settings > Domains.
